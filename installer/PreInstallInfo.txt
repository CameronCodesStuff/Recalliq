RecallIQ - AI-Powered Local File Search
========================================

RecallIQ indexes your documents and lets you search them using
natural language queries. Everything runs locally on your machine.

WHAT GETS INSTALLED:
  - RecallIQ application files
  - Start Menu shortcuts
  - Desktop shortcut (optional)

WHAT GETS CREATED:
  - %LOCALAPPDATA%\RecallIQ\  (settings and database)
  - models\ folder for ONNX AI model (optional)
  - tessdata\ folder for OCR data (optional)

RUNTIME DEPENDENCIES:
  If not already present, the installer will offer to download
  and install the following (requires internet):
  - .NET 8 Desktop Runtime (~55 MB)
  - Windows App SDK Runtime (~40 MB)

SUPPORTED FILE TYPES:
  PDF, DOCX, TXT, Markdown, PNG, JPG, TIFF

SYSTEM REQUIREMENTS:
  - Windows 10 version 1809 or later
  - 64-bit processor
  - 4 GB RAM minimum (8 GB recommended)

PRIVACY:
  RecallIQ runs entirely offline. No data is sent anywhere.
  No telemetry. No analytics. No cloud dependencies.
