# User Manual

## Getting Started

Launch RecallIQ. The Dashboard shows indexed documents, search statistics, and recent activity.

## Adding Folders

1. Open Settings (Ctrl+Shift+S)
2. Click Add Folder, select a directory
3. Click Save Settings

RecallIQ indexes all supported files in watched folders including subdirectories.

## Searching

1. Open Search (Ctrl+2)
2. Type a natural language query
3. Press Enter

Results show file name, matching paragraph, relevance score, and file path.

## Supported Formats

PDF, DOCX, TXT, Markdown, PNG, JPG, TIFF

| Shortcut | Action |
|----------|--------|
